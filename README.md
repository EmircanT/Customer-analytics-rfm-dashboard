 Müşteri Analitiği ve RFM Segmentasyon Projesi

Bu proje, FLO e-ticaret veri seti kullanılarak müşteri davranışlarını analiz etmek amacıyla geliştirilmiştir. Projede Python tabanlı veri analizi ile Excel üzerinde oluşturulan interaktif dashboard birlikte kullanılmıştır.

Proje Kapsamı

Projede aşağıdaki analizler gerçekleştirilmiştir:

- Veri temizleme ve dönüştürme işlemleri
- Feature engineering süreçleri
- Müşteri yaşam süresi hesaplama
- Recency, Frequency, Monetary (RFM) analizi
- Müşteri segmentasyonu
- Churn (müşteri terk) analizi
- Kanal bazlı gelir analizi
- KPI hesaplamaları
- İnteraktif Excel dashboard geliştirilmesi

 Kullanılan Teknolojiler

- Python
- Pandas
- NumPy
- Excel
- Pivot Table
- Slicer
- Dashboard Tasarımı

 Hesaplanan Metrikler

- Total Customer Value
- Average Order Value
- Customer Lifetime
- Recency
- Frequency
- Monetary
- Churn Rate

 Dashboard İçeriği

Excel dashboard içerisinde aşağıdaki analizler yer almaktadır:

- Kanal bazlı gelir dağılımı
- Segment bazlı gelir analizi
- Segment bazlı müşteri dağılımı
- Ortalama sipariş değeri analizi
- Churn analizi
- KPI kartları
- Dinamik filtreleme (Slicer)

 Elde Edilen İçgörüler

Bu proje sayesinde:

- Yüksek değerli müşteri segmentleri belirlendi
- Churn riski taşıyan müşteriler analiz edildi
- Satış kanallarının performansı karşılaştırıldı
- Müşteri davranışları incelendi
- Gelir optimizasyonuna yönelik içgörüler elde edildi
